# 🚀 CodeAlpha_Tasks

This repository contains all the projects I built during my **CodeAlpha Web Development Internship**. Each folder is a separate, full-stack task with its own README, setup instructions, and live demo.

## 🚀 Run Any Task Locally

1. Clone the repo

   ```
   git clone https://github.com/arshitakamboz-ui/CodeAlpha_Tasks.git
   cd CodeAlpha_Tasks
   ```

2. Go into the task folder you want to run

   ```
   cd Task1_EcommerceStore   # or Task2_SocialMedia / Task3_ProjectManager
   ```

3. Follow the setup steps in that task's own README — each one has its own dependencies and `.env` configuration.
